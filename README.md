Static site for Sector3 Racing, hosted with GitHub Pages.

## Stack
- Plain HTML and CSS
- Small vanilla JS for header behavior and mobile nav at /assets/js/site.js
- Adobe Fonts via Typekit
- GitHub Pages hosting
